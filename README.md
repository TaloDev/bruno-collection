# Bruno collection

This repo hosts a [Bruno](https://usebruno.com) request collection for [Talo](https://trytalo.com) - the open source game backend.

## Get started

Visit the [Bruno collection page](https://docs.trytalo.com/docs/http/bruno-collection) inside the Talo docs to learn how to use the collection.

## Interactive explorer

An API explorer is available [here](https://docs.trytalo.com/bruno-collection.html).

## Import into Bruno

1. Open Bruno and click **Import Collection**.
2. Paste the repository URL: `https://github.com/TaloDev/bruno-collection.git`
3. Choose a location to save the collection locally.
4. Click **Clone**.

## API docs

More guides, troubleshooting tips and request/response examples are available in the [Talo docs](https://docs.trytalo.com/docs/http/authentication).
